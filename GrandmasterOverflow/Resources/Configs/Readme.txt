To play the game you will need to install the 32-bit (x86) version of the "Visual C++ Redistributable Packages for Visual Studio 2013" from here:
http://www.microsoft.com/en-gb/download/details.aspx?id=40784

To toggle fullscreen on/off or change the window size look in the "Settings.ini" file.

To exit the game press the 'Esc' key.
